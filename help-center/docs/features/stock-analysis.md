# Stock Analysis

RightStockAI offers two powerful analysis methods to evaluate Indian stocks comprehensively.

## Analysis Methods Overview

### Traditional Analysis
Time-tested methodologies combining technical indicators and fundamental metrics.

[Learn More about Traditional Analysis](./traditional-analysis)

### AI Analysis
Advanced machine learning algorithms analyzing 56+ financial metrics.

[Learn More about AI Analysis](./ai-analysis)

## How to Analyze a Stock

### Step 1: Search for Stock

1. Go to [Stock Details](https://www.rightstockai.com/stocks)
2. Use the search bar to find stocks by:
   - Company name (e.g., "Reliance Industries")
   - Stock symbol (e.g., "RELIANCE")
   - Sector or industry

### Step 2: Choose Analysis Method

From the [Analysis Dashboard](https://www.rightstockai.com/analysis):

**For Fundamental Focus:**
- Choose Traditional Analysis
- Comprehensive financial metrics
- Bankruptcy risk assessment
- Earnings quality analysis

**For AI-Powered Insights:**
- Choose AI Analysis
- Pattern recognition
- Predictive analytics
- Confidence-scored recommendations

### Step 3: Review Analysis Results

Both methods provide:
- **Investment Score**: 0-10 rating scale
- **Detailed Metrics**: Financial and technical indicators
- **Risk Assessment**: Volatility and risk metrics
- **Recommendations**: Buy, Hold, or Sell guidance
- **Price Targets**: Short and long-term predictions

## Stock Details Page

Access comprehensive stock information at [Stock Details](https://www.rightstockai.com/stocks):

### Overview Tab
- Current price and daily change
- Market cap and volume
- 52-week high/low
- P/E ratio and dividend yield

### Charts Tab
View advanced charts at [Advanced Charts](https://www.rightstockai.com/chart):
- Multiple timeframes (1D, 1W, 1M, 3M, 6M, 1Y, 5Y)
- Technical indicators overlay
- Drawing tools
- Chart type options (Candlestick, Line, Bar)

### Fundamentals Tab
- Financial statements
- Key ratios
- Company profile
- Management information

### Technicals Tab
- RSI (Relative Strength Index)
- MACD (Moving Average Convergence Divergence)
- Bollinger Bands
- Moving Averages (50-day, 200-day)
