**Altman Z-Score**
- Bankruptcy prediction model
- Financial distress indicator
- Corporate health measurement
- Score interpretation:
  - **Z > 2.99**: Safe zone
  - **1.81 < Z < 2.99**: Grey zone
  - **Z < 1.81**: Distress zone

**Piotroski F-Score**
- Fundamental strength assessment (0-9 scale)
- Quality of earnings evaluation
- Financial position analysis
- Score interpretation:
  - **8-9**: Strong fundamentals
  - **5-7**: Moderate fundamentals
  - **0-4**: Weak fundamentals

**Beneish M-Score**
- Earnings manipulation detection
- Accounting red flags identification
- Financial reporting quality
- Score interpretation:
  - **M > -1.78**: Possible manipulation
  - **M < -1.78**: Likely accurate reporting
