## Daily Usage Limits Explained

### What Counts as a "Request"?

**AI Stock Analysis Request:**
- Running AI analysis on one stock = 1 request
- Same stock analyzed multiple times = multiple requests

**Traditional Analysis Request:**
- Running traditional analysis on one stock = 1 request
- Each new analysis = 1 request

**AI News Analysis Request:**
- Viewing AI analysis for one stock's news = 1 request
- Each stock's news analysis = separate request

**Portfolio Analysis Request:**
- Analyzing one portfolio = 1 request
- Re-analyzing same portfolio = new request

**Chart Request:**
- Opening/loading a chart = 1 request
- Viewing same chart (cached) = no additional request
- Changing timeframe/indicators = may count as new request

**Data Export Request:**
- Exporting one analysis to PDF/Excel = 1 request
- Each export = 1 request

### When Do Limits Reset?

**Reset Time:** Every day at 12:00 AM IST

**Example:**
- Used 5 AI analyses by 11:00 PM today
- At 12:00 AM, you get 5 new requests
- Previous day's usage doesn't carry over

### What Happens When You Hit Limits?

**On Free Plan:**
- You'll see "Daily limit reached" message
- Wait until 12:00 AM IST for reset
- Or upgrade to Pro/Premium for more
- Previously generated analyses remain accessible

**Pro Tip:** If you consistently hit Free plan limits, Pro plan is perfect for you!
